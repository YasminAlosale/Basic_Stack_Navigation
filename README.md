#  مشروع مكتبة قصص الأنبياء 


##  وصف المشروع 
تطبيق "مكتبة قصص الأنبياء" هو تطبيق تعليمي تم تطويره باستخدام Flutter. يهدف المشروع إلى تطبيق مفهوم التنقل بين الشاشات (Stack Navigation) وتمرير البيانات بين الواجهات بشكل سلس ومنظم.

##  رحلة المستخدم 
1. الواجهة الرئيسية: تعرض قائمة بأسماء الكتب (قصص الأنبياء) بتصميم جذاب وألوان هادئة.
2. واجهة التفاصيل: عند الضغط على أي قصة، ينتقل التطبيق لعرض تفاصيل القصة المختارة باستخدام نظام الـ Navigator.
3. التفاعل عند العودة: بعد الانتهاء من القراءة والعودة، تظهر رسالة تنبيه (SnackBar) في الأسفل تؤكد العودة بنجاح.
